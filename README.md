# voting-app

## Requirements
- Go https://go.dev/doc/install

## Recommendations
- Air
```
go install github.com/cosmtrek/air@latest
```

## Run
```
go mod init mywebsite.tv/name
```

If using air
```
air
```

If not
```
go run cmd/main.go
```

## On your browser
- Navigate to http://localhost:4000
